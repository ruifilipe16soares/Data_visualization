# Data_visualization
Creating a pipeline to get dynamic dashboards from a MySQL Database.
3 Keypoints:
-Designed and implemented a real-time data architecture to provide healthcare management professionals with a free, data-driven decision-support solution.
-Built an event-driven pipeline capturing data from the main database (MySQL), streaming through Kafka and indexing into Elasticsearch.
-Developed Python-based data transformation scripts and visualized insights through Kibana dashboards, all deployed via Docker.
